## Decisions

- GitHub является единственным источником истины.
- После каждого утвержденного изменения Planner автоматически формирует GitHub Task.
- GitHub Worker не принимает архитектурных решений.
- GitHub Worker изменяет только то, что явно указано в GitHub Task.
- Один документ отвечает только за одну область ответственности.
- История проекта ведется отдельными файлами по дням.
