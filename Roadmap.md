# Roadmap

## Этап I. Основание

✓ Definition

✓ Architecture

→ Document Standard

• Утвержден единый стандарт документов ViktorOS.

• Все документы создаются в универсальном Markdown.

• Все документы в процессе разработки отображаются только как исходный Markdown.

• Режим предпросмотра (Rendering / Writing) при разработке не используется.

• Документы должны одинаково отображаться в любых Markdown-редакторах.

• Используется единый минималистичный визуальный язык.
